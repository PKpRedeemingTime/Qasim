Clients and Servers

Computers connected to the Web are called clients and servers.

  • Clients are the typical Web user's Internet-conneted devices.
  • Servers are computers that store webpages, sites or apps.
So What Actually Happens?
  • The browser goes to the DNS Server and finds the real address.
  • The browser sends an HTTP request message to the server.
  • If the server approves the client's request the server sends the client a "200 OK" message.
  • The browser assembles the small chunks into a website.
